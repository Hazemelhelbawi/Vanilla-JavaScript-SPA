# Vanilla JavaScript SPA

A simple **Single Page Application (SPA)** built with **Vanilla JavaScript** without using any frameworks like React or Vue.

This project demonstrates how SPA routing works under the hood using native browser APIs.

---

## Live Demo

You can run the project locally to see how routing works without a page reload.

---

## Features

- Vanilla JavaScript (No frameworks)
- Client-side routing
- Dynamic page loading
- History API navigation
- Clean and simple project structure
- Beginner-friendly code

---

## What You Will Learn

This project helps you understand:

- What a **Single Page Application (SPA)** is
- How **client-side routing** works
- How frameworks like **React Router** work internally
- Using the **History API**
- Loading HTML pages dynamically
